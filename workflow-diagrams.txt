Data Flow Example:
══════════════════
Trigger:
  Every day at 8:00 AM (Cron: 0 8 * * *)

Get Open Tasks (HTTP GET):
  URL: http://vikunja.vikunja.svc.cluster.local:3456/api/v1/tasks/all?filter=done=false
  Returns: Array of task objects

Prepare Task Data:
  taskList: {{ $json }}
  taskCount: {{ $json.length }}

Basic LLM Chain Input:
  "Hier ist die Liste aller offenen Aufgaben:
  
  [JSON array of tasks]
  
  Bitte erstelle eine Zusammenfassung in Deutsch mit:
  1. Gesamtanzahl der offenen Aufgaben
  2. Aufgaben nach Priorität gruppiert
  3. Heute fällige Aufgaben (2026-02-16)
  4. In den nächsten 3 Tagen fällige Aufgaben
  5. Überfällige Aufgaben"

CONNECTION TYPES EXPLAINED
═══════════════════════════

main:
  Standard data flow between nodes
  Carries JSON data from output to input
  Example: Chat Trigger → AI Agent

ai_languageModel:
  Connects LLM to agent or chain
  Provides language model capability
  Example: Ollama Chat Model → AI Agent

ai_outputParser:
  Connects output parser to agent
  Enforces output structure (JSON schema)
  Example: Structured Output Parser → AI Agent
