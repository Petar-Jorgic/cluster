WORKFLOW 2: Daily Task Summary
-------------------------------
Purpose: AI-generated summary of all open tasks in German
Trigger: Schedule (Cron: 0 8 * * * - Daily at 8:00 AM)
Model: llama3.1:8b-instruct-q4_0 @ http://ollama.ollama.svc.cluster.local:11434

Node Flow:
  Schedule Trigger (8 AM daily)
    → Get Open Tasks (HTTP GET from Vikunja)
    → Prepare Task Data
    → Basic LLM Chain (with Ollama Chat Model)
    → Format Summary Output
    → [Optional] Send Summary (webhook - disabled by default)

Summary Includes:
  • Total open tasks
  • Tasks grouped by priority (1-5)
  • Tasks due today
  • Tasks due in next 3 days
  • Overdue tasks

Vikunja API:

  Auth: Bearer tk_ba2a509cbe18e5368df76e9de325f55e12506358
